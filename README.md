Python programming language is an essential part of Data Science and it helps a learner in his journey as a data scientist. Being a good programmer helps a learner to prepare data better for analysis and allows to quickly deploy models. Following concepts are covered in this program

- Fundamentals of Python language
- Object-Oriented Programming and other advanced concepts
- Manipulating matrices using NumPy
- Data Wrangling with Pandas
- Visualizing Data using Matplotlib
